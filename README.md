# Лабораторная работа № 1. Разведочный анализ больших данных с Apache Spark

**Дисциплина:** Системы обработки больших данных (СОБД)

## Цель работы
Выполнить разведочный анализ (EDA) датасета большого объёма с помощью
Apache Spark (PySpark) на кластере Hadoop (HDFS + YARN) с использованием
табличного формата Apache Iceberg.

## Датасет
NYC Yellow Taxi Trip Data - данные о поездках жёлтого такси Нью-Йорка.
Источник: https://www.kaggle.com/datasets/elemento/nyc-yellow-taxi-trip-data
Объём: несколько гигабайт.
Основные признаки: время посадки и высадки, число пассажиров, дистанция
поездки, сумма поездки, чаевые, сборы, зоны посадки и высадки и др.

## Структура репозитория
- notebooks/1_Data_cleansing.ipynb - загрузка и очистка данных
  (типы признаков, обработка пропусков и выбросов).
- notebooks/2_Exploratory_analysis.ipynb - разведочный анализ
  (статистические показатели, распределения признаков, корреляции).
- scripts/ - вспомогательные скрипты.
- report/ - глава курсовой работы (15-20 страниц).

## Технологии
Apache Spark (PySpark), Hadoop HDFS, YARN, Apache Iceberg, Python, Jupyter.
